## 🚀 **Getting Started in 3 Minutes**

### **Prerequisites**
```bash
✅ Python 3.8+ (3.10+ recommended)
✅ 4GB RAM minimum (8GB+ for local models)
✅ Windows/macOS/Linux support
🎯 Ollama (optional - for local AI models)
```

### **Lightning-Fast Installation**

#### **Option 1: Clone & Run (Recommended)**
```bash
# Get The Oracle
git clone https://github.com/southernpitbull/Oracle.git
cd Oracle

# Install dependencies (auto-installs everything you need)
pip install -r requirements.txt

# Launch The Oracle
python main.py
```

#### **Option 2: Local AI Models (Privacy-First)**
```bash
# Install Ollama for local models
# Visit: https://ollama.ai

# Pull some powerful models
ollama pull llama3.1:8b      # Great all-around model
ollama pull codellama:13b    # Specialized for coding
ollama pull mistral:7b       # Fast and efficient

# Launch The Oracle - local models work immediately!
python main.py
```

### **First-Time Setup Magic**

1. **🔑 API Keys (Optional)**
   - Only needed for cloud providers (OpenAI, Anthropic, etc.)
   - Local models work immediately with zero configuration
   - Add keys via: `Settings → API Configuration`

2. **🎨 Personalize Your Experience**
   - Choose your preferred theme (dark/light/auto)
   - Set up your favorite models
   - Configure keyboard shortcuts

3. **🚀 Start Creating**
   - Select a provider and model
   - Type your first message
   - Experience the future of AI interaction!
